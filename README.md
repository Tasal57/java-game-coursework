Project Title

Java 2D Physics-Based Platformer Game

Overview

This project is a 2D game developed in Java as part of a university coursework module.
The game uses a physics engine to simulate gravity, collisions, and movement.
Players navigate through levels, avoid obstacles, and interact with game objects.

Key Features

Physics-based movement and collisions

Player-controlled character with keyboard input

Platforms, walls, and interactive objects

Enemies and hazards

Scoring and game state management

Multiple levels

Technologies Used

Java

Object-Oriented Programming (OOP)

Physics Engine (as provided in coursework)

IntelliJ IDEA

Git & GitHub

Game Mechanics

Horizontal and vertical player movement

Jumping and gravity-based physics

Collision detection with platforms and obstacles

Health / score tracking

Enemy interactions
